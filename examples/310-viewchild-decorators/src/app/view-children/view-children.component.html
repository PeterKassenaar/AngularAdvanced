Decorator @ViewChildren: a list of customers

Below is the component app-customer-detail inside an *ngFor. So it appears 4 times. It is fetched in the TypeScript. We now use the @ViewChildren decorator, this returns a list of matches. In this case a QueryList<CustomerDetailComponent>.
